# JWT Token Generator(C++)

Библиотека (DLL) для генерации и криптографической проверки JWT-токенов с использованием алгоритма **HMAC-SHA256**.
Написана на чистом C++ с использованием Windows Cryptography API (CNG), не требует сторонних библиотек (например, OpenSSL).

## Файлы в проекте
* `JWToken.cpp` — исходный код библиотеки (ядро генерации и проверки).
* `test.cpp` — консольное приложение-интерфейс для проверки работы DLL.

## Как использовать библиотеку (.dll)
Библиотека экспортирует две функции в стиле `C`:

1. **Генерация токена:**
`void GenerateToken(const char* parameters, const char* secret_key, char* out_buffer, int buffer_size);`

2. **Верификация токена:**
`int VerifyToken(const char* token, const char* secret_key);`
Возвращает `1`, если подпись верна, и `0`, если токен подделан/изменен.
